# Gestor Master Frontend

Frontend web do Gestor Master.

Este repositorio implementa a experiencia web administrativa e operacional. A fonte oficial de requisitos e decisoes fica no repositorio irmao `../gestor-master-docs`.

## Primeira entrega

Gestao de Acesso MVP:

- Tela de login.
- Tela de Ativacao de Conta.
- Fluxo de recuperacao de senha.
- Administracao minima de usuarios.
- Administracao minima de equipes.
- Administracao minima de projetos.

## Design

- Fonte central: `../gestor-master-docs/docs/design/visual-identity.md`
- Tokens locais: `design/tokens.json`
- Implementacao web: `src/styles.css`
